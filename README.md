# Stored XSS Attack Demonstration & Mitigation Lab

A Flask-based cybersecurity project demonstrating how Stored Cross-Site Scripting (Stored XSS) attacks occur in web applications and how they can be prevented using secure coding practices.

## Features

- User Login using Flask Sessions
- Comment System with SQLite
- Stored XSS Demonstration
- Secure & Vulnerable Mode Toggle
- Output Escaping using Jinja2
- Modern Responsive UI
- Clear Comments Functionality

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- Jinja2

## Security Concepts

- Stored Cross-Site Scripting (XSS)
- Output Encoding
- Session Management
- Secure Template Rendering
- OWASP Secure Coding Practices
